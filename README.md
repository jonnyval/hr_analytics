# Прогнозирование оттока сотрудников (HR Analytics)

## Описание проекта
Проект посвящен разработке предиктивной модели машинного обучения для HR-департамента. Главная цель — научиться прогнозировать вероятность увольнения сотрудников и выявить ключевые факторы, влияющие на отток кадров, чтобы вовремя принимать меры по удержанию ценных специалистов.

## Задачи исследования
1. Провести разведочный анализ данных (EDA) и исследовать «портфель увольняющихся».
2. Подготовить и очистить данные (обработка пропусков, поиск неявных дубликатов).
3. Обучить ML-модель для предсказания факта увольнения сотрудника.
4. Оценить качество предсказания с помощью кастомной метрики (SMAPE).

## Стек технологий
* **Язык:** Python
* **Библиотеки:** Pandas, Matplotlib, Seaborn, Scikit-learn
* **Алгоритмы и метрики:** Decision Tree / Random Forest, SMAPE (Symmetric Mean Absolute Percentage Error)

## Ключевые инсайты (EDA)
В ходе глубокого исследования портфеля увольняющихся были обнаружены интересные закономерности:
* **Занятость:** Неожиданно выяснилось, что сотрудники с низкой загрузкой (уровнем занятости) увольняются чаще всех. Отсутствие понятных задач при необходимости физического присутствия в офисе приводит к быстрому выгоранию.
* Проведено сравнение влияния других атрибутов (зарплата, стаж, отдел) на вероятность ухода сотрудника.
